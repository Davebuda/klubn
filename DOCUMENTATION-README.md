### Measure Everything:
```
Track these metrics from day 1:
- Daily Active Users (DAU)

- Resale transaction volume
- Revenue per user
- Viral coefficient (k)

Month 12: 40% DAU
```

### Get Feedback Early:
```
Before building each feature:
1. Show mockup to 10 users
2. Ask: "Would you use this?"
3. Ask: "What's confusing?"
4. Iterate design based on feedback
5. THEN start coding

Saves weeks of development time!
```

## 🔧 Technical Stack Recommendations

### Already Using (Your Current Stack):
```
✅ Backend: .NET 8, C# 12, ASP.NET Core
✅ Database: SQLite (for now), EF Core 9
✅ API: GraphQL (HotChocolate 13)
✅ Auth: JWT Bearer tokens
✅ Frontend: React 18, TypeScript, Vite
✅ State: Zustand
✅ Styling: Tailwind CSS
```

### Add for New Features:

**Media Storage:**
- AWS S3 or Cloudinary (for photos/videos)
- Cost: ~€50-200/month depending on usage

**Real-Time:**
- SignalR or Socket.io (for live chat/updates)
- Already compatible with .NET stack

**Mobile Wallet:**
- PassKit (Apple Wallet) - Free
- Google Wallet API - Free
- Just need Apple Developer account (€99/year)

**Live Streaming (Later):**
- Mux (easiest, $$$), Cloudflare Stream ($$), or AWS IVS ($)
- Delay until Phase 5 (Month 9-10)

**Push Notifications:**
- Firebase Cloud Messaging (FCM) - Free
- Apple Push Notification service (APNs) - Free
